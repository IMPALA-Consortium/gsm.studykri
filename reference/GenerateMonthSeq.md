# Generate Complete Month Sequence

Generate Complete Month Sequence

## Usage

``` r
GenerateMonthSeq(start_yyyymm, end_yyyymm)
```

## Arguments

- start_yyyymm:

  numeric. Start month in YYYYMM format

- end_yyyymm:

  numeric. End month in YYYYMM format

## Value

data.frame with MonthYYYYMM column
